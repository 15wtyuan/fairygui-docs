ThreeJS

运行Demo

  1. 从GitHub Clone或者直接下载FairyGUI-threejs-example。
  2. 运行npm run build。
  3. Demo的UI工程在UIProject下。结合UI工程和例子代码可以了解到FairyGUI的使用方法。

加入FairyGUI到你的项目

npm i fairygui-three

编译源码

  1. 从GitHub Clone或者直接下载FairyGUI-threejs。
  2. 运行npm run build。
  3. 可以运行npm run test测试。
